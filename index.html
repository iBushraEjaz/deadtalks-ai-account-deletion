DeadTalks AI – Account & Data Deletion

DeadTalks AI allows users to create an account using Firebase Authentication. The information stored may include:

  • Email address (if used for sign-in)
  • User ID (UID)
  • Authentication tokens

How users can delete their account inside the app

Users can delete their account directly from the DeadTalks AI app by following these steps:

  1. Open the app and go to the Profile section.
  2. You will see two options: Logout and Delete Account.
  3. Tap on Delete Account.
  4. The app will ask the user to type DELETE to confirm.
  5. After typing “DELETE” and pressing Delete Account, the user’s account is permanently removed.

When the account is deleted:

  • The Firebase Authentication user account is permanently deleted.
  • The user's UID and all authentication data are removed from Firebase.
  • No personal data associated with the user is retained.

Manual account deletion request

If a user cannot access the app or prefers manual deletion, they may request account removal by contacting:
